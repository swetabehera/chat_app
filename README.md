# Chat App

Built as a part of learning TypeScript concepts, it is a chat application with rooms.

## Features :
- Create usernames to chat
- Create chat rooms
- Send messages
- Receive messages

## Tech Stack :
- TypeScript
- CSS (BootStrap)
- Node.js
- ExpressJS
- React.js
- React hooks
